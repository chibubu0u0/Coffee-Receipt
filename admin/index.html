Quick Import

快速導入資料

貼上網址、頁面文字或 CSV，先產生欄位預覽，再套用到表單或直接寫入 Firestore。v9 採「來源資料模式」：AI 只抽取來源明確寫出的資訊，不自動評分、不做人格/情緒/色彩判斷。

1. AI 資料抽取

建議先貼「頁面文字」再按 AI 解析文字。AI 只會整理來源明確出現的欄位；若來源沒有寫酸質、甜感等 0–5 數字，會留空，不會自動判斷。

2. 貼網址解析

適合 Best of Panama / 烘豆商商品頁。若網站是動態頁，系統會盡量讀取公開 HTML，並保留來源網址。

3. 貼頁面文字

從商品頁複製公開文字貼上來，AI / 規則解析都會優先使用這裡的內容。

4. 貼 CSV 批量解析

第一列放欄位名稱，例如 name,country,region,process,cuppingScore,published。
